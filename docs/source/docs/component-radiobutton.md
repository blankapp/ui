title: RadioButton

Single selection controls

## Examples

```jsx
<RadioButton.Group>
  <RadioButton text={'Option1'} checked={true} />
  <RadioButton text={'Option2'} />
</RadioButton.Group>
```

## API

### Props
Name | Description | Type | Optional value | Default
--- | --- | --- | --- | ---
`CircleComponent` | - | element | - | -
`text` | - | string | - | Empty string ('')
`value` | - | string | - | -
`checked` | - | bool | - | `false`
