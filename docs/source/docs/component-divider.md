---
title: Divider
---
Used to separate distinct content sections

## Examples

```jsx
<Divider />
```

## Variations

### Orientation

```jsx
<Divider orientation={'vertical'} />
<Divider orientation={'horizontal'} />
```

## API

### Props

Name | Description | Type | Optional value | Default
--- | --- | --- | --- | ---
`orientation` | - | enum | `vertical`, `horizontal` | `horizontal`
