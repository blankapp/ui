Small count controls

## Examples

```jsx
<Badge text={'99+'} />
```

## Types

### dot

```jsx
<Badge type={'dot'} />
```

### text

```jsx
<Badge type={'text'} text={'99+'} />
```

## API

### Props

Name | Description | Type | Optional value | Default
--- | --- | --- | --- | ---
`children` | - | string, element | - | -
`type` | - | enum | `dot`, `text` | `dot`
`text` | - | string | - | Empty string ('')
