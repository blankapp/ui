## 示例

```jsx
<Switch />
```

## 状态

### 禁用

```jsx
<Switch disabled />
```

### 选中

```jsx
<Switch checked />
```

## API

### 属性

名称 | 描述 | 类型 | 可选值 | 默认值
--- | --- | --- | --- | ---
`disabled` | 是否处于禁用状态 | bool | - | `false`
`checked` | 是否处于选中状态 | bool | - | `false`
