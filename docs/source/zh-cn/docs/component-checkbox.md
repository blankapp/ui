title: CheckBox

多选控件

## 示例

```jsx
<CheckBox text={'CheckBox'} checked={true} />
```

## API

### 属性
名称 | 描述 | 类型 | 可选值 | 默认值
--- | --- | --- | --- | ---
`children` | - | string, element | - | -
`text` | 显示的文本 | string | - | 空字符串 ('')
`checked` | 是否处于选中状态 | bool | - | `false`
